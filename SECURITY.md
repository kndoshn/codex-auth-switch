# Security Policy

## Supported Versions

This project is not yet versioned with long-term support branches.
Please report issues against the latest commit on `main`.

## Reporting a Vulnerability

Do not open a public issue for vulnerabilities involving authentication data, session files, or account switching behavior.

Please report vulnerabilities privately by email:

- `kndoshn@gmail.com`

Include the following when possible:

- affected version or commit
- reproduction steps
- impact assessment
- whether authentication files or tokens may be exposed

I will acknowledge receipt and investigate before any public disclosure.
